# E-learning-platform-DBMS

## 📌 Description

This project implements a normalized database (up to 3NF) for an E-learning platform.

## 🛠️ Technologies Used

* MySQL (XAMPP)
* phpMyAdmin

## 📂 Files

* elearning.sql → Database schema and data
* normalization.pdf → UNF to 3NF process

## 🚀 How to Run

1. Install XAMPP
2. Start Apache & MySQL
3. Open http://localhost/phpmyadmin
4. Create database `elearning`
5. Import elearning.sql

## ✅ Features

* Fully normalized database (3NF)
* Proper relationships using foreign keys
* Many-to-many handled using Enrollment table
